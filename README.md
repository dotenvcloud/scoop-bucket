# DotEnv Scoop Bucket

Scoop manifests for DotEnv CLI.

## Installation

```powershell
scoop bucket add dotenv https://github.com/dotenvcloud/scoop-bucket
scoop install dotenv-cli
```

## Available Apps

- `dotenv-cli` - DotEnv CLI for secure environment variable management

Visit [https://dotenv.cloud/docs/cli](https://dotenv.cloud/docs/cli) for more information.
